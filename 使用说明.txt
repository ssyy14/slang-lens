Slang Lens — 英语俚语翻译
============================

📦 安装方法
-----------
双击「安装.bat」→ 自动安装到电脑并创建桌面快捷方式

🚀 使用方法
-----------
1. 双击桌面「Slang Lens」图标
2. 点右上角齿轮 ⚙，填入 DeepSeek API Key
3. 输入英文俚语/梗，点翻译即可

🔑 获取 API Key
----------------
打开 https://platform.deepseek.com
注册 → 充值（很便宜，几块钱用很久）

📋 功能
--------
- 字面直译 + 真实含义 + 文化背景 + 用法 + 例句
- IPA 音标 + 朗读发音
- 手机/平板/电脑全适配
- 查询缓存，重复秒开

💡 提示
--------
- API Key 只存在你的浏览器里，不会泄露
- 可以直接把「Slang Lens.html」发给任何人用
- 这是个纯前端网页，不依赖任何服务器
