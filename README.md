# Slang Lens — 英语俚语翻译

理解英语，不止翻译。

输入英语俚语、典故、网梗，获得的不只是生硬翻译，还有：
- 📌 **字面直译** — 每个词本来是什么意思
- 🔥 **真实含义** — 本地人真正想表达什么
- 📖 **文化背景** — 出处、演变、使用人群
- 💬 **用法说明** — 什么场景用、什么语气
- ✏️ **双语例句** — 英文原句 + 中文翻译

## 快速开始

1. 下载 `index.html`
2. 用浏览器打开
3. 点击右上角齿轮 ⚙，填入 [DeepSeek API Key](https://platform.deepseek.com)
4. 输入英文俚语，点翻译

## 技术

- 纯 HTML + CSS + JS，单文件，零依赖
- DeepSeek API 驱动翻译 + 文化解释
- localStorage 缓存，重复查询秒开
- 响应式设计，手机/平板/桌面都适配

## 隐私

- API Key 只存储在浏览器 localStorage
- 所有翻译请求直接从浏览器发往 DeepSeek API
- 不经过任何第三方服务器
- 你的 Key 你做主

## 示例

| 输入 | 类型 |
|------|------|
| `spill the tea` | 俚语 — 爆料、八卦 |
| `no cap` | 俚语 — 不骗你、说真的 |
| `ghost someone` | 俚语 — 突然消失、玩失踪 |
| `rizz` | 网梗 — 魅力、撩人能力 |
| `break a leg` | 典故 — 祝好运 |
| `TL;DR` | 缩写 — 太长不看 |
| `I like coffee` | 普通表达 — 也正常翻译 |
